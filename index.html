API

/new

Requests a new username, and it's availability info.

{
    "username": "",
    "availability": {
        "github": true,
        "linkedin": true,
        "dot-com": true,
        "facebook": true,
        "twitter": true
    }
}

/availability/:service?username=<username>

Requests availability for a specific service and a given username.

{
    "service": true
}
